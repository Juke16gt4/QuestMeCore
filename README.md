# QuestMeCore

QuestMeCore は、**キャラクター音声作りに特化したマルチプラットフォーム音声モジュール**です。  
「音声設定ボタン」ひとつで外部アプリと接続可能。ユーザーは声質や声色を自由に調整し、**自分だけのキャラクター音声を創造する楽しさ**を体験できます。

---

## 🎮 特徴: ゲーム性に富んだ音声作り

- **音声設定ボタンで即接続**  
  外部アプリとの連携はシンプル。ボタンひとつで接続し、キャラクター音声を自由に呼び出せます。

- **声質の自由な調整**  
  - 速さ  
  - トーン  
  - ビブラート  
  - 途切れ声  
  → スライダーで直感的に操作可能

- **声色の最終確認**  
  - サンプル発話で「キャラクターらしさ」を即チェック  
  - 確定ボタンで保存し、外部アプリに反映

- **ゲーム性のある体験**  
  - プレイヤーが「声を作る」楽しみを提供  
  - キャラクターの個性を音声で表現  
  - 開発者にとっては「声を遊びに変える」新しい表現手段

---

## ✨ 新しい変更点

- **VoiceSettingsView**  
  - スライダーで声質を調整  
  - サンプル発話で即確認  
  - UIはシンプルで安心感のあるデザイン  

- **ConfirmationView**  
  - 声質設定を引き継ぎ、声色を最終確認  
  - ヘルプボタン付きツールチップで初心者にも優しい設計  

---

## 🚀 QuestMeCore の良さ

- **安心感**: 医療現場の知見を活かし、ユーザーが安心して操作できる体験を提供  
- **拡張性**: 外部アプリに簡単に接続でき、ゲームや学習アプリに統合可能  
- **再現性**: UI/UX 設計はすべて「再現可能な儀式」として明文化され、チーム開発でも迷いがない  
- **楽しさ**: キャラクター音声を「作る」こと自体が遊びになる
