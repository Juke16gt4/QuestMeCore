# QuestMeCore
QuestMeCore is a core SDK for healthcare and self-reflection apps, combining speech synthesis with emotion profiles. Provides character-level highlighting and modular voice components for SwiftUI.「QuestMeCore は、音声合成と感情表現を組み合わせたヘルスケア／自己内省アプリ向けのコア SDK です。 SwiftUI と AVSpeechSynthesizer を活用し、文字単位のハイライトや感情プロファイルを提供します。」
